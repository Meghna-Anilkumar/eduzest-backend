# 🚀 Eduzest Backend

## 📌 Overview
Eduzest is a full-featured e-learning platform designed to provide a complete online learning experience. The backend powers authentication, course management, payments, assessments, and real-time communication.

This project is built using **Node.js, Express, MongoDB, and TypeScript**, following **Clean Architecture and Repository Pattern** for scalability and maintainability.

---

## ✨ Features

- 🔐 OTP-based Authentication & Google OAuth
- 👩‍🎓 Role-based Access (Student, Instructor, Admin)
- 📚 Course Creation & Management
- 🎥 Video Upload & Lesson Handling (AWS S3)
- 💳 Secure Payments & Subscriptions (Stripe)
- 📝 Assessments & Exams
- 🏆 Leaderboard System
- 💬 Real-time Group Chat (Socket.IO)
- 🎟️ Coupon & Offer Management
- 🛡️ JWT-based Authorization

---

## 🏗️ Architecture

This project follows:

- Repository Pattern
- Modular Folder Structure
- Separation of Concerns

---

## 🛠️ Tech Stack

- Node.js
- Express.js
- TypeScript
- MongoDB & Mongoose
- JWT Authentication
- Stripe API
- AWS S3
- Socket.IO
- Redis (for caching / chat optimization)
